# Investor Behavior Survey Analysis (SQL + Excel)

### Objective
Analyze investor risk appetite, trading preferences, and investment trends using SQL.

### Key Queries
1. Age group vs preferred trading type  
2. Average investment by risk appetite  
3. Experience vs monthly investment  
4. Platform popularity  

### Key Insights
• Younger investors prefer intraday trading  
• Medium-risk investors contribute highest average investment  
• Groww & Zerodha are top trading platforms
